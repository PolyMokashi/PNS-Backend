---

## Installation
### 1. Clone the repository:
```bash
git clone https://github.com/PolyMokashi/PNS-Backend.git
cd PNS-Backend
```

### 2. Install dependencies:
```bash
npm install
```

### 3. Start the server:
```bash
npm start
```
